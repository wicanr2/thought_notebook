# 思考模型索引

這裡收錄從新聞、文章、論文與專題筆記中抽象出來的判斷框架。它不是原始材料庫，而是「多篇筆記反覆出現後，值得獨立保存的思考工具」。

## 什麼時候新增模型

當同一個判斷框架在不同筆記中反覆出現，且能跨題材重用時，再整理成模型頁。暫時性的新聞線索先放在根目錄 README 的「目前主題線索」或各分類年度索引，不急著膨脹成模型。

建議新增門檻：

1. 至少能連回 2–3 篇既有筆記。
2. 能提供一組可操作的問題或檢查清單。
3. 不是單純概念名詞，而是能幫助判斷、比較或拆解材料。
4. 有明確的適用邊界與容易誤用的地方。
