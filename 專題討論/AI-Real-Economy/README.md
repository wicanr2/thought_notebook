# AI Real Economy｜AI 進入實體經濟

這個專題整理 AI 從模型、軟體與資料中心，進一步進入製造、能源、物流、國防、機器人、工業設備與資本市場時，需要面對的部署問題。

核心問題不是「AI 會不會有用」，而是：

> 當 AI 要改造實體世界時，真正卡住的是模型、資料、能源、認證、流程整合、供應鏈，還是資本架構？

## 可重用問題

- AI 應用是在數位流程，還是實體流程？
- 它需要哪些專有資料？資料在誰手上？
- 它是否受物理法則、工安、認證或法規約束？
- 它是在雲端執行，還是必須 edge deployment？
- 它能否嵌進既有 workflow，而不是要求客戶重建整套流程？
- 它需要多少能源、算力與基礎設施？
- 它適合哪種資本：VC、growth equity、project finance、private credit、投資級債、政府資本？
- 如果要放大十倍，真正卡住的是技術、能源、認證、供應鏈，還是融資？
