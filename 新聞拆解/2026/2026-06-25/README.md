## 今日觀察

今天最有價值的共同主題是：**技術系統的瓶頸常常不在表面產品，而在被視為背景的約束。**

- 電力系統的瓶頸不是只有裝置容量，而是冷卻、河水溫度、尖峰需求與法規邊界。
- 晶片供應鏈的瓶頸不是只有晶片本身，而是微影設備與盟友政策協調。

這兩篇剛好形成一組對照：一篇是物理環境約束，一篇是地緣政治約束。兩者都提醒我，分析產業新聞時不要只看「誰推出了什麼」，要問：**誰控制了不可替代的邊界條件？**
