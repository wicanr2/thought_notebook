## 置頂文章要點｜第一性原理的陷阱

- 文章出處：[Vincent Cheng-Wen Yu 余鎮文｜第一性原理的陷阱：當你用工程思維誤讀人性](https://vincentcwyu.substack.com/p/ad1)
- 筆記：[文章拆解｜第一性原理的陷阱：工程思維誤讀人性](./文章拆解/2026/2026-06-25/01-第一性原理的陷阱-工程思維誤讀人性.md)
- 主題整理：[思考模型｜社會科學的第一性原理](./思考模型/社會科學的第一性原理.md)
- 核心提醒：第一性原理不是「免除脈絡成本」的通行證；在社會系統裡，過度簡化常常不是接近真理，而是刪掉真正重要的制度、權力與人性約束。
- 最重要的轉念：社會科學中的第一性通常不是定律，而是約束，例如資源稀缺、資訊不對稱、激勵不相容、交易成本。
- 分析新聞時要特別檢查：我是否找錯主詞？真正被滿足的是使用者需求，還是掌權者、付費者、監管者或資本市場的恐懼與誘因？
- 好用模型：工程系統追求唯一真解；社會系統更常追求堅固性（robustness）——即使假設偏離、信任下降、參與者套利，也不要立刻崩潰。
- 方法論警語：拆掉看似不合理的制度柵欄前，先搞清楚它原本防止的是哪種災難。
- 喜歡的引用：「拆掉柵欄前，請先搞清楚它為什麼建在那裡。」——引自 [Vincent Cheng-Wen Yu 余鎮文〈第一性原理的陷阱：當你用工程思維誤讀人性〉](https://vincentcwyu.substack.com/p/ad1)，概念源自 Chesterton's Fence。
