## 筆記格式

每篇拆解通常包含：

1. 原文資訊
2. 為什麼選這篇
3. 一句話拆解
4. 文章架構地圖
5. 第一性原理拆解
6. 寫法與敘事策略
7. 可抽象成的思考模型
8. 可放進資料庫的筆記
9. 仍不確定的地方

## 調閱方式

- 想看某天選了哪些文章：進入該日期資料夾的 `README.md`。
- 想看完整分析：點進日期資料夾下的單篇 `.md`。
- 想找主題：先看本頁的「目前主題線索」，之後會逐步擴成更完整的索引。
